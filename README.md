# Fuck macOS

## Installation

1. Unzip and copy files to the App folder of your Onion OS SD card

```
root
└── App
    └── Fuck_macOS
        ├── config.json
        ├── launch.sh
        ├── logo.png
        └── script.sh
```

2. Insert the SD card into the Miyoo device and turn it on

3. Chose "Fuck macOS" and click button `A`
